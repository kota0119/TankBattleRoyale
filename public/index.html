Tank Battle Royale

      【操作方法】

      タンクの移動 : 方向キー / WASDキー

      弾丸の発射 : スペースキー

      地雷の設置 : Mキー

      ステルス状態の切り替え : Tキー

      Mapの表示・非表示切り替え : Pキー

      Infoの表示・非表示切り替え : Iキー

      【アイテム効果】

      : 一定時間タンクの回復速度が上昇する

      : 一定時間タンクのダメージを軽減する

      : 一定時間タンクの移動速度が上昇する

      : 一定時間タンクの攻撃力が上昇する
